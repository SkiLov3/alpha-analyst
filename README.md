# Alpha Analyst 📈

Institutional-grade stock picking tool designed for medium-to-long term wealth preservation and growth.

## 🏦 Investment Strategy: "The Fidelity Model"
This tool follows a **Quality at a Reasonable Price (QARP)** methodology, favored by top-tier institutional desks. It prioritizes companies with strong balance sheets, high capital efficiency, and sustainable growth trajectories.

### Scoring Factors
- **PEG Ratio (40%)**: Ensuring growth isn't overvalued.
- **Return on Equity (30%)**: Measuring management's ability to generate profit from shareholder equity.
- **Debt-to-Equity (20%)**: Filtering for financial stability and solvency.
- **Trend/Momentum (10%)**: Validating the entry point relative to the 200-day moving average.

## 🚀 Getting Started
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the morning report:
   ```bash
   python picker.py
   ```

## 📊 Diversification logic
The algorithm automatically selects the top-performing stocks across multiple sectors (Tech, Healthcare, Financials, Consumer, Industrials, etc.) to ensure a balanced portfolio risk profile.
